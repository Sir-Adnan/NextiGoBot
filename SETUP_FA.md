# راهنمای نصب و راه‌اندازی ربات

## مرحله ۱: دریافت توکن ربات

1. به [@BotFather](https://t.me/botfather) در تلگرام پیام دهید
2. دستور `/newbot` را ارسال کنید
3. یک نام برای ربات انتخاب کنید (مثال: فروشگاه من)
4. یک یوزرنیم برای ربات انتخاب کنید که با `bot` تمام شود (مثال: `myshop_bot`)
5. توکن ربات را کپی کنید (شبیه: `123456789:ABCdefGHIjklMNOpqrsTUVwxyz`)

## مرحله ۲: دریافت آیدی عددی خودتان

1. به [@userinfobot](https://t.me/userinfobot) پیام دهید
2. آیدی عددی خود را کپی کنید (مثال: `123456789`)

## مرحله ۳: تنظیم فایل محیطی

1. فایل `.env.example` را کپی کرده و نام آن را `.env` بگذارید:
```bash
cp .env.example .env
```

2. فایل `.env` را باز کنید و مقادیر را وارد کنید:
```env
BOT_TOKEN=توکن_ربات_شما
ADMIN_ID=آیدی_عددی_شما
CARD_NUMBER=شماره_کارت_شما
CARD_HOLDER_NAME=نام_دارنده_کارت
```

## مرحله ۴: نصب Docker

### ویندوز
1. [Docker Desktop](https://www.docker.com/products/docker-desktop/) را دانلود کنید
2. نصب کنید و سیستم را ری‌استارت کنید

### لینوکس
```bash
curl -fsSL https://get.docker.com -o get-docker.sh
sudo sh get-docker.sh
sudo systemctl start docker
sudo systemctl enable docker
```

## مرحله ۵: اجرای ربات

در پوشه پروژه دستور زیر را اجرا کنید:

```bash
docker-compose up -d
```

ربات شروع به کار خواهد کرد!

## مشاهده لاگ‌ها

```bash
docker-compose logs -f
```

## توقف ربات

```bash
docker-compose down
```

## ری‌استارت ربات

```bash
docker-compose restart
```

## به‌روزرسانی ربات

```bash
docker-compose down
docker-compose build --no-cache
docker-compose up -d
```

## استفاده از ربات

### برای ادمین:

1. `/start` را به ربات ارسال کنید
2. دکمه "⚙️ پنل ادمین" را بزنید
3. محصول اضافه کنید
4. برای هر محصول، اکانت اضافه کنید
5. وقتی مشتری خرید کرد، رسید پرداخت را چک کنید
6. سفارش را تایید یا رد کنید

### برای مشتری:

1. `/start` را به ربات ارسال کنید
2. محصول مورد نظر را انتخاب کنید
3. دکمه "💳 خرید" را بزنید
4. مبلغ را به کارت واریز کنید
5. رسید را برای ربات ارسال کنید
6. بعد از تایید، اطلاعات اکانت دریافت می‌کنید

## عیب‌یابی

### ربات استارت نمی‌شود
- بررسی کنید توکن درست وارد شده باشد
- بررسی کنید Docker در حال اجرا باشد

### ربات پیام نمی‌دهد
- با دستور `/start` ربات را راه‌اندازی کنید
- بررسی کنید ADMIN_ID درست باشد

### خطای دیتابیس
```bash
docker-compose down -v
docker-compose up -d
```

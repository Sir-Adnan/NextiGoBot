# 🔥 راهنمای استفاده از ایموجی‌های پریمیوم تلگرام

## مشکل: چرا ایموجی‌های پریمیوم نمایش داده نمی‌شوند؟

ایموجی‌های پریمیوم تلگرام به دو صورت کار می‌کنند:

### 1. ایموجی‌های استاندارد با افکت
- این ایموجی‌ها برای **کاربران پریمیوم** انیمیشن دارند
- برای کاربران عادی، به صورت ایموجی استاندارد نمایش داده می‌شوند
- ✅ **در حال حاضر استفاده می‌شود**

### 2. Custom Emoji (نیاز به Telegram Premium)
- این ایموجی‌ها فقط برای کاربران پریمیوم قابل مشاهده هستند
- نیاز به Custom Emoji ID دارند
- ⚠️ **محدودیت: فقط اگر ربات Premium باشد**

## راه‌حل فعلی ✅

ربات شما **در حال حاضر از ایموجی‌های زیبا استفاده می‌کنه**! بذارید توضیح بدم:

### قبل (بدون ایموجی):
```
سلام علی!
به ربات فروشگاه خوش آمدید.
```

### بعد (با ایموجی‌های پریمیوم):
```
✨ سلام علی! 🔥

🛍️ به فروشگاه ما خوش آمدید 🎁
از منوی زیر می‌تونید محصولات رو مشاهده و خرید کنید 🚀
```

## چطور ببینم کار می‌کنه؟

### تست 1: شروع ربات
```
/start
```
باید استیکر خوش‌آمدگویی + پیام با ایموجی‌های زیبا ببینید

### تست 2: مشاهده محصولات
```
دکمه "🛍️ محصولات"
```
باید ایموجی‌های 📦 و 💰 رو در توضیحات ببینید

### تست 3: پنل ادمین (اگر ادمین هستید)
```
دکمه "⚙️ پنل ادمین"
```
باید منوی کامل با ایموجی‌ها رو ببینید:
- ➕ افزودن محصول
- 📦 مدیریت محصولات
- 📋 سفارشات در انتظار
- 📊 آمار فروش

## افزودن ایموجی‌های بیشتر

### 1. ویرایش فایل `bot/emojis.py`:

```python
class PremiumEmojis:
    # ایموجی جدید اضافه کنید
    DIAMOND = "💎"
    ROBOT = "🤖"
    LIGHTNING = "⚡"
```

### 2. استفاده در پیام‌ها:

```python
text = f"{PremiumEmojis.DIAMOND} محصول ویژه!"
```

## استفاده از Custom Emoji (پیشرفته)

اگر می‌خواید Custom Emoji واقعی استفاده کنید:

### گام 1: دریافت Custom Emoji ID

1. یک استیکر پک Custom Emoji بسازید
2. از [@Stickers](https://t.me/Stickers) استفاده کنید
3. Custom Emoji ID رو دریافت کنید

### گام 2: اضافه کردن به کد

```python
from telegram import MessageEntity
from telegram.constants import MessageEntityType

# ارسال پیام با Custom Emoji
entities = [
    MessageEntity(
        type=MessageEntityType.CUSTOM_EMOJI,
        offset=0,
        length=1,
        custom_emoji_id="5368324170671202286"
    )
]

await update.message.reply_text(
    "🔥 پیام با افکت!",
    entities=entities
)
```

### گام 3: فعال‌سازی در تنظیمات

در فایل `.env`:
```env
USE_PREMIUM_EMOJIS=true
```

## استیکرهای انیمیشن‌دار

ربات الان از استیکرها هم استفاده می‌کنه:

### استیکر خوش‌آمدگویی
وقتی `/start` می‌زنید، یک استیکر خوش‌آمدگویی ارسال می‌شه

### استیکر موفقیت
وقتی سفارش تکمیل می‌شه، استیکر جشن ارسال می‌شه

### افزودن استیکرهای بیشتر

1. استیکر رو به ربات خودتون بفرستید
2. File ID رو دریافت کنید:

```python
async def get_sticker_id(update, context):
    if update.message.sticker:
        sticker_id = update.message.sticker.file_id
        print(f"Sticker ID: {sticker_id}")
```

3. اضافه کنید به `bot/stickers.py`:

```python
MY_STICKER = 'CAACAgQAAxkBAAI...'
```

## عیب‌یابی

### مشکل: هیچ ایموجی نمایش داده نمی‌شود

**راه‌حل:**
```bash
# چک کردن تنظیمات
cat .env | grep PREMIUM

# باید ببینید:
USE_PREMIUM_EMOJIS=true
```

### مشکل: استیکر ارسال نمی‌شود

**راه‌حل:**
1. Check import ها:
```python
from bot.stickers import StickerManager
```

2. Check handler:
```python
await StickerManager.send_welcome_sticker(update, context)
```

### مشکل: خطا در import

**راه‌حل:**
```bash
# نصب مجدد پکیج‌ها
pip install -r requirements.txt

# یا با Docker
docker-compose build --no-cache
docker-compose up -d
```

# 🚀 راه‌اندازی سریع (۵ دقیقه)

<div dir="rtl">

## مرحله ۱: دریافت توکن ربات (۲ دقیقه)

1. به [@BotFather](https://t.me/botfather) در تلگرام پیام دهید
2. دستور `/newbot` را بزنید
3. نام و یوزرنیم ربات را وارد کنید
4. توکن را کپی کنید

## مرحله ۲: دریافت آیدی خود (۳۰ ثانیه)

1. به [@userinfobot](https://t.me/userinfobot) پیام دهید
2. آیدی عددی خود را کپی کنید

## مرحله ۳: تنظیم پروژه (۱ دقیقه)

```bash
# کپی فایل تنظیمات
cp .env.example .env

# ویرایش با notepad یا هر ادیتور
notepad .env
```

در فایل `.env` این مقادیر را وارد کنید:
```
BOT_TOKEN=توکن_ربات_شما
ADMIN_ID=آیدی_عددی_شما
CARD_NUMBER=شماره_کارت_شما
CARD_HOLDER_NAME=نام_شما
```

## مرحله ۴: اجرای ربات (۳۰ ثانیه)

### با Docker:
```bash
docker-compose up -d
```

### بدون Docker:
```bash
pip install -r requirements.txt
python run_local.py
```

## ✅ تمام!

حالا ربات شما آماده است! به ربات در تلگرام `/start` بزنید.

## 📝 گام‌های بعدی

1. وارد پنل ادمین شوید: دکمه "⚙️ پنل ادمین"
2. محصول اضافه کنید: "➕ افزودن محصول"
3. اکانت اضافه کنید: "مدیریت محصولات" → انتخاب محصول → "➕ افزودن اکانت"
4. آماده فروش! 💰
